2015-10-06

记 2015 年生日

  • 回顾
  • 生日
thought

日子过的还真是快，转眼间 2015 年已经过去一大半了，距离乔帮主去世也已经 4 年了，正巧生日，也就回顾一下这大半年来的生活吧。

年初定下的目标

回顾年初定下的全年目标，已经完成了一多半了。特别是其中有些意想不到的事情让自己很满足，也很庆幸。距离 2015 年底还有两个月的时间，抓紧时间完成全年的目标。

水比赛

2015 年水了一些比赛，拿到了一些不足挂齿的奖项。

美乐送

美乐送上线了，虽然第一版有点挫，但是第二版会好太多。

SegmentFault 专栏

SegmentFault 首篇文章首日浏览破千，3天破6K。发布当天被 SegmentFault 官方微信图文推送，浏览3300+，第二天被官微推送，想想还真是有点小激动呢。第一次写文章，连自己的朋友圈都没有分享，仅凭那惹人骂的标题就获得了这样的“小成就”，真是受宠若惊。

GitHub 第一个 Contribution

为 Amaze UI 修复了一个小 BUG，获得了 2.4.0 版本的感谢。这是我真正意义上在 GitHub 的第一个贡献。

和一流的人才共事

遇到了一些一流的优秀人才，在 GitHub 上斩获 2300+ star 的高贵 iOS 工程师 @Draveness，前端特别溜的 @proudrain，游戏天才萌学弟 @于京平……和优秀的人在一起工作才能变得更加优秀。很庆幸，在年轻的时候能够遇到这么一群优秀的人才。

年轻真好

大三了，不同的人有着不同的路要走。很幸运，自己在大学的这段时间里没有太大压力，一直在做着自己喜欢的事情，一直保持乐观积极的心态，像小孩子一样生活，只想说年轻真好。

one more thing

另外希望早日入手 Mac，哈哈哈 :)
